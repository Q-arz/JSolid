# Contributing to JSolid

## Setup
```bash
npm install
npm run build
npm run test
```

## Guidelines
- Mantén el código claro y tipado (TS donde aplique)
- Añade tests (unit/integration) con cada feature
- Documenta en `docs/es` y `docs/en`
- Usa PRs pequeños y enfocados

## Estructura
- `src/` compilador/generador/deployer/runtime
- `docs/es` y `docs/en` documentación bilingüe
- `examples/` ejemplos ejecutables
- `test/` unit/integration/E2E
