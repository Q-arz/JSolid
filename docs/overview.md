# JSolid Docs - Overview

JSolid permite escribir contratos en JavaScript/TypeScript y generar Solidity listo para compilar y desplegar.

- 3 formas de escribir contratos:
  - Nivel A: Objetos (JSON-like)
  - Nivel B: Clases JavaScript (ES6)
  - Nivel C: TypeScript con decoradores
- Incluye plantillas OpenZeppelin (ERC20/721/1155, Ownable, Pausable, AccessControl) y patrón Diamond.
- Compila con `solc` y despliega con `ethers`.
