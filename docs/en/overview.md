# JSolid Docs - Overview

JSolid lets you write smart contracts in JavaScript/TypeScript and generate Solidity ready to compile and deploy.

- 3 authoring modes:
  - Level A: Object (JSON-like)
  - Level B: JavaScript Classes (ES6)
  - Level C: TypeScript with decorators
- Templates: ERC20/721/1155, Ownable, Pausable, AccessControl, and Diamond pattern.
- Compile with `solc` and deploy with `ethers`.
